HTML Tags

This is the list of HTML Tags:

  1. <html> and </html>: These tags define the beginning and end of an HTML document.
  2. <head> and </head>: These tags contain information about the HTML document, such as the title, meta tags, and links to CSS and JavaScript files.
  3. <body> and </body>: These tags contain the content of the HTML document, such as text, images, and links.
  4. <h1> to <h6>: These tags define headings of different levels, with <h1> being the most important and <h6> being the least important.
  5. <p>: This tag defines a paragraph of text.
  6. <br>: This tag inserts a line break.
  7. <img>: This tag inserts an image into the document.
  8. <a>: This tag defines a hyperlink.
  9. <ul> and </ul>: These tags define an unordered list.
  10. <ol> and </ol>: These tags define an ordered list.
  11. <li>: This tag defines a list item.
  12. <table> and </table>: These tags define a table.
  13. <tr> and </tr>: These tags define a row in a table.
  14. <td> and </td>: These tags define a cell in a table.
  15. <form> and </form>: These tags define a form.
  16. <input>: This tag defines an input field, such as a text field, checkbox, or button.
  17. <label>: This tag associates a label with an input field.
  18. <select> and </select>: These tags define a drop-down list.
  19. <option>: This tag defines an option in a drop-down list.
  20. <textarea> and </textarea>: This tag defines a multi-line text field.
  21. <div> and </div>: These tags define a generic division in the HTML document.
  22. <span> and </span>: These tags define a generic inline element in the HTML document.
